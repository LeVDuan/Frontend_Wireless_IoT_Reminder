# Frontend Wireless IoT Reminder

## Project Description

The Frontend Wireless IoT Reminder project is developed by LeVDuan. This project uses the ESP32 WROOM 32 module to connect to the COM port on the computer and control it through a web interface.

## Technologies Used

- **ESP32 WROOM32 module**: This is the IoT module used in this project.
- **WebSerial API**: This API is used to connect to the COM port.
- **Git**: Git is used for source code version management.
- **Next.js**: This is the web application development framework used in this project.
- **TypeScript**: TypeScript is the main programming language used in this project.

## Installation

To install and run this project, follow these steps:

1. Clone the project from GitHub:

   ```bash
   git clone https://github.com/LeVDuan/Frontend_Wireless_IoT_Reminder.git
   ```

2. Install the dependencies:

   ```bash
   npm install
   ```

   Or

   ```bash
   yarn install
   ```

3. Run the project:

   ```bash
   npm run dev
   ```

   Or

   ```bash
   yarn dev
   ```

## Usage

After installing and running the project, you can access the web interface through your browser and connect to the COM port via the WebSerial API.
